This is a client and firmware distribution for the GoodFET, an
open source adapter for the MSP430, CC2430, SPI Flash, and
various other chips.

You should probably be using the latest code from the git repo, which
you can grab like so:

git clone https://github.com/travisgoodspeed/goodfet
